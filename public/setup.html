⚙️ Environment Variables (Disarankan)

Cara lebih aman: set variabel di server agar tidak perlu setup ulang setelah restart.

SMTP_HOST=mail.perusahaan.com
SMTP_PORT=587
SMTP_USER=noreply@perusahaan.com
SMTP_PASS=password_anda
SMTP_FROM=noreply@perusahaan.com
DIRECTOR_EMAIL=direktur@perusahaan.com
HRD_EMAIL=hrd@perusahaan.com
DIRECTOR_NAME=Nama Direktur
COMPANY_NAME=PT. Perusahaan Anda
SERVER_URL=http://103.x.x.x:3000
PORT=3000

Simpan di file .env dan install dotenv, atau set langsung di VPS.
